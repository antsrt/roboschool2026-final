# Робошкола 2026 — Финал

Этот репозиторий будет дополняться по мере появления новых материалов и файлов.

## Описание задания

Полное описание финального задания находится в документе:

- [docs/task-description.md](docs/task-description.md)

## Окружение

Окружение предоставляется в виде подготовленных Docker-контейнеров.

## Требования к ОС

Для запуска контейнеров нужна одна из следующих ОС:

- Ubuntu 20.04 или 22.04
- Windows 10 и новее с установленным WSL (не рекомендуется)

## Подготовка системы

Перед началом работы выполните шаги из следующих гайдов:

1. Docker: установка
   - https://docs.docker.com/engine/install/ubuntu/
2. Docker: настройка после установки
   - https://docs.docker.com/engine/install/linux-postinstall/
3. Для работы видеокарты NVIDIA внутри контейнеров: `nvidia-container-toolkit`
   - https://docs.nvidia.com/datacenter/cloud-native/container-toolkit/latest/install-guide.html
